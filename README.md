# Mobile Animation Maker (Gradio)

இது பைத்தான் மற்றும் Gradio-வை பயன்படுத்தி உருவாக்கப்பட்ட ஒரு எளிய அனிமேஷன் கருவி. இதை Termux (Android) அல்லது கணினியில் இயக்கலாம்.

## சிறப்பம்சங்கள்
- படங்களை (Images) வீடியோவாக மாற்றலாம்.
- FPS (வேகம்) மாற்றிக்கொள்ளலாம்.
- MP4 அல்லது AVI ஃபார்மேட்டில் டவுன்லோட் செய்யலாம்.
